# wiki/concepts/

Concept and topic pages — ideas, techniques, frameworks, and definitions worth keeping a stable page for.

A concept page typically has:

- A short definition / TL;DR
- Key distinctions (what it is *not*)
- Links to source material in `raw/`
- Cross-links to related concepts, projects, and people

Filenames: kebab-case (`prompt-caching.md`, `bayesian-optimization.md`).
