## Concepts

- [Effective prompting (umbrella)](concepts/effective-prompting.md) — index page for the prompting techniques surfaced from the *Claude prompting guide*.
- [Few-shot prompting](concepts/few-shot-prompting.md) — show, don't (just) tell: include an example of the desired output.
- [Chain-of-thought prompting](concepts/chain-of-thought-prompting.md) — ask the model to think step-by-step and explain its reasoning.
- [Role prompting](concepts/role-prompting.md) — assign the model a persona to focus content and tone.
- [Iterative prompt refinement](concepts/iterative-prompt-refinement.md) — refine with named deltas, not "make it better".
- [LLM conversation statelessness](concepts/llm-statelessness.md) — Claude doesn't carry context across separate conversations by default.
- [Socratic-tutoring prompt pattern](concepts/socratic-tutoring-prompt-pattern.md) — two-mode system prompt: Socratic tutor for concepts, direct assistant for artifacts.
