# content/

Content pipeline — drafts and published pieces flow through here. Wired up in Part 2 of the second-brain setup.

## Expected layout (from Part 2)

- `ideas/` — seeds, prompts, things you might want to write about
- `drafts/` — works-in-progress
- `published/` — frozen final versions, with metadata about where they appeared

Source material from `raw/` and synthesis from `wiki/` feed into pieces here. Until Part 2 sets up the pipeline, this folder can stay empty.
