# raw/pdfs/

General-purpose PDFs that don't fit a more specific source folder. If a PDF is an arXiv paper, put it in `arxiv-papers/`; if it came from Zotero, put it in `zotero/`.

## What belongs here

- Books and book chapters
- Slide decks and lecture handouts
- Manuals, datasheets, and white papers
- Government / institutional reports
- Scanned documents

## Where to export from

- **Browser** — `Print → Save as PDF` from any web page when you want a frozen copy.
- **Publisher sites** — direct download is best when available.
- **Scanned docs** — Apple Notes scanner, Adobe Scan, or a real scanner; OCR with `ocrmypdf` if the text isn't selectable.
- **Slides** — Google Slides `File → Download → PDF`, Keynote `File → Export To → PDF`.

Filename convention: `YYYY-author-or-org-shorttitle.pdf` (e.g. `2024-anthropic-responsible-scaling.pdf`). Run OCR before saving if the file is image-only.
