# raw/

Your source documents. **Claude does not modify anything in this folder** — it is read-only from the assistant's perspective.

Each subfolder corresponds to one source/medium you bring into the vault. Drop the original file in as-is; processing, summaries, and synthesis live in `wiki/` instead.

## Subfolders

- `articles/` — web articles, blog posts, newsletters
- `claude-exports/` — exported conversations from claude.ai
- `podcast-transcripts/` — transcripts of podcast episodes
- `youtube-transcripts/` — transcripts of YouTube videos
- `voices-memos/` — voice memos and their transcripts
- `slack-exports/` — Slack workspace/channel exports
- `jira/` — Jira issue exports
- `email-archives/` — exported email threads (mbox, eml, etc.)
- `pdfs/` — general PDFs that don't fit a more specific source
- `arxiv-papers/` — academic papers from arXiv (and other preprint servers)
- `zotero/` — Zotero library exports and attachments
- `notes/` — personal notes exported from other apps (Apple Notes, Notion, etc.)

Add new subfolders here when you onboard a new source. Each new subfolder should get its own README explaining what belongs in it and where to export it from.
