# raw/arxiv-papers/

Academic papers from arXiv and other preprint servers. Keep it focused on preprints / research papers — generic PDFs go in `pdfs/`, Zotero-managed papers go in `zotero/`.

## What belongs here

- arXiv preprints (cs.*, stat.ML, math.*, etc.)
- Other preprints: bioRxiv, medRxiv, SSRN, OpenReview submissions
- Conference / journal versions of the same papers when you want to keep both

## Where to export from

- **arXiv** — `https://arxiv.org/pdf/<id>` for the PDF, `https://arxiv.org/abs/<id>` for the landing page.
- **arxiv-sanity / Semantic Scholar** — follow the PDF link.
- **OpenReview** — `Download PDF` from the forum page.
- **`arxiv` CLI** — `arxiv download <id>` for scripted bulk pulls.

Filename convention: `YYYY-firstauthor-shorttitle.pdf` (e.g. `2017-vaswani-attention.pdf`). Optionally save a companion `.bib` or `.md` next to each paper with citation info and your one-paragraph summary.
