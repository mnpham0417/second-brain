# raw/articles/

Web articles, blog posts, and newsletter issues saved as Markdown (preferred) or PDF.

## What belongs here

- Long-form blog posts and essays
- Newsletter issues worth keeping
- News articles you want to reference later

## Where to export from

- **Readwise Reader** — `Export → Markdown` from a document, or use the bulk Markdown export from Settings.
- **Instapaper** — `Settings → Export → Download .html`, then convert to Markdown.
- **Pocket** — `Settings → Export → HTML file`, then convert.
- **Browser** — extensions like *MarkDownload* or *Obsidian Web Clipper* save the page directly as `.md`.
- **Substack/newsletters** — forward to a service like Readwise/Kindle, or "Save as PDF" from the browser.

Filename convention: `YYYY-MM-DD-slug.md` (date the article was published or saved).
