# journal/

Daily notes — one file per day. Wired up in Part 2 of the second-brain setup.

## Convention

- Filename: `YYYY-MM-DD.md` (e.g. `2026-04-24.md`).
- Free-form: what happened, what you're thinking about, decisions, open questions.
- Cross-link liberally to `wiki/` pages so concepts/projects/people accumulate context over time.

Daily journal entries are the authoritative timeline of *your* day; `wiki/log.md` is the timeline of *Claude's* edits to the wiki.
